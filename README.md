# BWS-FR Fixpack

BWS-FR Fixpack est une compilation de correctifs destinés au BWS-FR.  
Il est basé sur le fonctionnement du EE-Mod-Fixpack.

- RelieveWizardSlayer.tp2 v1.7

- Tweak Antlogoy v15
